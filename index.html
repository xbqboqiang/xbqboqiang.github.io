deque容器 #include < deque>
双端数组，迭代器支持随机访问
与vector区别：
vector对于头插删除效率低，数据量越大越低，deque对头插删除速度更快
vector访问元素的速度比deque快
内部工作原理：
内部有个中控器，维护每个缓冲区的地址，使得使用deque时像一片连续的空间，缓冲区内存放真实数据

#1512. 好数对的数目

给你一个整数数组 nums 。

如果一组数字(i, j) 满足 nums[i] == nums[j] 且 i < j ，就可以认为这是一组 好数对 。

返回好数对的数目。

示例 1：

输入：nums = [1, 2, 3, 1, 1, 3]
输出：4
解释：有 4 组好数对，分别是(0, 3), (0, 4), (3, 4), (2, 5) ，下标从 0 开始
示例 2：

输入：nums = [1, 1, 1, 1]
输出：6
解释：数组中的每组数字都是好数对
示例 3：

输入：nums = [1, 2, 3]
输出：0

来源：力扣（LeetCode）
链接：https ://leetcode-cn.com/problems/number-of-good-pairs
